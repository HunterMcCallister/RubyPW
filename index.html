Description & History

Ruby is a programming language focused on simplicity and productivity. Created by Yukihiro “Matz” Matsumoto in 1995. Matz made Ruby by combining his favorite parts of other languages. For example, Ruby’s block is heavily inspired by Lisp and other functional languages. With features like this and inspiration from many different languages Ruby gained massive popularity around 2006, and while that popularity has gone down over the years it continues to keep relevant with software like “Ruby on Rails”.

Installation

Ruby can be installed on macOS, Linux, or Windows. Below is the recommended way to install Ruby and run Ruby programs using the Ruby interpreter.

macOS (Homebrew)

macOS includes an older version of Ruby. It is recommended to install a newer version using Homebrew:

# Install Ruby:
brew install ruby

# Check the version:
ruby -v

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install ruby-full
ruby -v

Windows

Install Ruby using the official RubyInstaller:

  • RubyInstaller for Windows

Interpreter

Ruby programs are executed with the ruby interpreter:

ruby program.rb

Ruby also includes the gem package manager for installing libraries:

gem install <package-name>
