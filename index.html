Description & History

Ruby is a programming language focused on simplicity and productivity. Created by Yukihiro “Matz” Matsumoto in 1995. Matz made Ruby by combining his favorite parts of other languages. For example, Ruby’s block is heavily inspired by Lisp and other functional languages. With features like this and inspiration from many different languages Ruby gained massive popularity around 2006, and while that popularity has gone down over the years it continues to keep relevant with software like “Ruby on Rails”.

Installation

Ruby can be installed on macOS, Linux, or Windows. Below is the recommended way to install Ruby and run Ruby programs using the Ruby interpreter.

macOS (Homebrew)

macOS includes an older version of Ruby. It is recommended to install a newer version using Homebrew:

# Install Ruby:
brew install ruby

# Check the version:
ruby -v

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install ruby-full
ruby -v

Windows

Install Ruby using the official RubyInstaller:

  • RubyInstaller for Windows

Interpreter

Ruby programs are executed with the ruby interpreter:

ruby program.rb

Ruby also includes the gem package manager for installing libraries:

gem install <package-name>

Simple Programs

To run each of these programs except Sort Numbers, all the user has to do is type and enter: 

ruby <filename>

Hello World:

puts "Hello, world!"

Print Range:

print "what is the range of numbers you want printed?\n"
print "starting number: "
startingNumber = gets
print "ending number: "
endingNumber = gets
if startingNumber > endingNumber 
	print "the starting number is greater than the ending number\n"
else 
	for i in startingNumber..endingNumber do
		print i
	end 
end

To run Sort Numbers the user must type and enter:

ruby <filename> [number]...

Sort Numbers:

print "what are the numbers that you want sorted?\n"
values = gets.split(' ')
values = values.sort 
print "here are the numbers sorted: "
for value in values do
	print value, " "
end
print "\n"

More Complex Programs

To run Guessing Game, all the user has to do is type and enter: 

ruby <filename>

Guessing Game:

value = Integer(rand(100) + 1)
print "guess of what number I am thinking of between 1-100\n"
print "guess: "
guess = Integer(gets)

while 1 
	if guess > value 
		print "the guess is greater than the value\n"
		print "guess: "
		guess = Integer(gets)
	elsif guess < value
		print "the guess is less than the value\n"
		print "guess: "
		guess = Integer(gets)
	else 
		print "the guess is correct. good job!\n"
		break;
	end 
end

To run Sign In Or Log In , the user must follow these steps:

Step 1 - Check which version of ruby they are using:

ruby -v

Step 2 - Go to Sqlite3 Gem Install and find the appropriate sqlite3 version for their ruby version

Step 3 - Type and run the command:

gem install sqlite3 -v 'VERSION'

(where VERSION is the appropriate version of sqlite3 for your version of ruby)

Step 4 - Execute Create Table only once by typing and entering:

ruby  <filename for Create Table>

Create Table:

require "sqlite3"

db = SQLite3::Database.new "values.db"

db.execute("create table userInfo (username varchar(50), password varchar(50))")

Step 5 - Execute Sign In or Log In as many times as you want by typing and entering:

ruby  <filename for Log In or Sign In>

Sign In or Log In:

require "sqlite3"
require "digest"

db = SQLite3::Database.new "values.db"

print "sign up or log in?\n"
print "0 - sign up\n"
print "1 - log in\n"
val = Integer(gets)

if val == 0
	print "username: "
	username = gets
	print "password: "
	password = gets
	rows = db.execute("select * from userInfo WHERE username=?", [username])
	alreadySignedUp = false
	if rows.length == 1 
		print "this username is already taken\n"
		alreadySignedUp = true
	end
	if !alreadySignedUp
		hashedPassword = Digest::SHA256.digest password
		db.execute("INSERT INTO userInfo (username, password) VALUES (?, ?)", [username, hashedPassword])
		print "signed up!\n"
	end	
elsif val == 1
	print "username: "
	username = gets
	print "password: "
	password = gets
	hashedPassword = Digest::SHA256.digest password
	rows = db.execute("select * from userInfo WHERE username=? AND password=?", [username, hashedPassword])
	if rows.length == 1
		print "logged in!\n"
	else
		print "username or password is incorrect\n"
	end
else 
	print "incorrect input, please either type 0 or 1\n"
end
